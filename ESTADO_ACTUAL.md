## Cómo Usar Ahora

### Opción 1: Inicio Rápido (5 minutos)

```bash
# 1. Clonar
git clone https://github.com/kndys123/KNDYS.git
cd KNDYS

# 2. Iniciar
chmod +x kndys.py && ./kndys.py

# 3. Ver módulos
kndys> show modules

# 4. Usar un módulo
kndys> use port_scanner
kndys(port_scanner)> set target scanme.nmap.org
kndys(port_scanner)> run
```

**Tiempo:** 5 minutos  
**Resultado:** Framework funcionando

### Opción 2: Aprender Completamente
